Ukraine - UKTZED

When declaring goods, all goods are subject to classification, i.e., codes are assigned to goods in accordance with the classification groups specified in the UKTZED. There are no goods that are not subject to classification. In the UKTZED, goods are systematized into sections, groups, commodity items, the names and numerical codes of which are unified with the Harmonized Commodity Description and Coding System. Module settings

To configure this module, select the Applications category, enter "UKTZED" in the search bar, and install the found module.

Next, go to Settings and activate developer mode. On the left, select Uktzed and click UPLOAD UKTZED.

After selecting this option, you get access to three methods of downloading/updating codes. Choose the method you need and click Upload. In a few minutes, you will receive information about the codes in your Odoo system. You can find it in the general menu by selecting UKTZED. Also from this menu, you can create a new desired code by simply clicking the CREATE button.
